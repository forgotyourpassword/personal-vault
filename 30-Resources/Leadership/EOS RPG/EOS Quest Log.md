# EOS Quest Log

Use this as the campaign tracker.

## Main Quests
- [ ] **Quest 1: Name the Beast**
  - Define EOS in your own words.
  - Name the 6 key components from memory.
  - Explain why businesses drift into complexity.
  - Reward: **20 XP**

- [ ] **Quest 2: The Six Kingdoms**
  - Write a 2-3 sentence explanation of Vision, People, Data, Issues, Process, and Traction.
  - For each one, give a real-world failure mode.
  - Reward: **30 XP**

- [ ] **Quest 3: Rock Setter's Oath**
  - Turn 5 vague goals into clear 90-day rocks.
  - Identify which are actually projects, habits, or wishes.
  - Reward: **25 XP**

- [ ] **Quest 4: Council of Numbers**
  - Draft a scorecard with 5-7 numbers for a team you understand.
  - Explain why each number matters.
  - Reward: **30 XP**

- [ ] **Quest 5: The Issue Pit**
  - Take 3 messy complaints and restate them as clean root issues.
  - For one of them, run a short IDS pass.
  - Reward: **35 XP**

- [ ] **Quest 6: Battle Rhythm**
  - Design a weekly meeting rhythm that does not suck.
  - Include issues list, scorecard review, rocks, and to-dos.
  - Reward: **35 XP**

## Side Quests
- [ ] Summarize one chapter from memory in under 150 words. **+10 XP**
- [ ] Teach one EOS idea to someone else in plain English. **+15 XP**
- [ ] Spot an EOS failure in a meeting and write down the diagnosis. **+10 XP**
- [ ] Draft a one-page core process. **+20 XP**
- [ ] Create a Harbor View-specific EOS experiment. **+20 XP**

## Daily/Weekly Drills
- [ ] 3 flash questions in one sitting. **+5 XP**
- [ ] 1 scenario diagnosis. **+10 XP**
- [ ] 1 plain-English explanation. **+5 XP**
- [ ] 1 application note in [[00-Inbox/Traction]]. **+10 XP**

## Campaign Milestones
- **50 XP:** Operator unlocked
- **120 XP:** Integrator unlocked
- **220 XP:** Implementer unlocked
- **350 XP:** Traction Warden unlocked

## Reward Ideas
When you clear a milestone, pair it with a real reward so your brain gets the message.
- Better coffee
- A new notebook or pen
- Book buying permission
- Time block for a favorite trail or ruck session
