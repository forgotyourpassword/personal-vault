# EOS Skill Tree

Think of these as mastery lanes. You do not need all of them at once, but eventually EOS asks for competence across the whole board.

## Tier 1: Foundations
### Vision Lore
**What it means:** understand where the business is going and how people align around it.

**Unlock by:**
- defining EOS in one paragraph
- explaining why lack of vision creates drag
- naming the 6 key components from memory

**Practical rep:** draft a simple vision statement for a team or project.

### People Sense
**What it means:** know the difference between liking someone and having them in the right seat.

**Unlock by:**
- explaining right person/right seat in plain English
- identifying one seat that would break if ownership is fuzzy
- describing what accountability looks like beyond job titles

**Practical rep:** sketch a seat-based accountability chart.

### Data Sight
**What it means:** move from opinions to numbers that show what is actually happening.

**Unlock by:**
- explaining why scorecards matter
- naming 3 numbers that would reduce confusion in a team
- distinguishing useful metrics from vanity metrics

**Practical rep:** create a 5-7 number scorecard for a real team.

## Tier 2: Operational Control
### Issue Hunting
**What it means:** get to the real problem instead of arguing with symptoms.

**Unlock by:**
- define IDS from memory
- show the difference between discussing and solving
- diagnose the root issue in 2 scenarios

**Practical rep:** rewrite a messy complaint as a clean issue statement.

### Process Forging
**What it means:** document the few core ways the business should consistently run.

**Unlock by:**
- explain why too much process backfires
- identify one core process worth documenting
- summarize the 3-7 step rule from your reading notes

**Practical rep:** draft a one-page core process.

### Traction Engine
**What it means:** make priorities visible and create real weekly rhythm.

**Unlock by:**
- explain a Rock without fluff
- explain why quarterly focus beats 18 simultaneous priorities
- describe what a good weekly meeting should accomplish

**Practical rep:** create 3 sample rocks and a weekly review loop.

## Tier 3: Leadership Application
### Meeting Command
**Unlock by:**
- redesign a bad standing meeting using EOS principles
- explain why agenda discipline matters
- identify what belongs in a Level 10 meeting and what does not

### Accountability Smithing
**Unlock by:**
- map responsibilities to seats, not personalities
- identify one recurring ownership gap in real life
- propose a cleaner structure in one page

### Simplicity Discipline
**Unlock by:**
- remove unnecessary complexity from a plan
- explain EOS in terms a normal employee would not hate
- reject one fake-sophisticated solution in favor of a simple one

## Legendary Perks
Earn these only with real-world use.
- **Harbor View Mapper:** you used EOS to clarify an actual Harbor View operating issue
- **Cadence Builder:** you created a repeatable review rhythm and stuck to it for 4 weeks
- **Issue Butcher:** you cut through noise and named the real issue quickly
- **Anti-Consultant Aura:** you can use EOS without sounding like a deck with legs

## Skill Tree Rule
If you cannot use the skill under pressure in a messy situation, it is still locked.
