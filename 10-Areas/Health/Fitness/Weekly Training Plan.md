# Weekly Training Plan

> Flexible 4-day week — pick any 4 mornings based on your schedule. Aim for no more than 2 strength days back to back.

---

## Recommended Weekly Structure

### Option A — Run-heavy week
| Day | Session | Duration |
|-----|---------|----------|
| Monday | Strength A (Upper Push) | 45–60 min |
| Tuesday | Easy Run (30–40 min) | 30–40 min |
| Thursday | Strength B (Upper Pull + Core) | 45–60 min |
| Saturday | Long-ish Run or Trail Run | 45–60 min |

### Option B — Balanced week
| Day | Session | Duration |
|-----|---------|----------|
| Monday | Strength A (Upper Push) | 45–60 min |
| Wednesday | Run + Ankle Mobility | 30–45 min |
| Friday | Strength B (Upper Pull + Core) | 45–60 min |
| Saturday | Bike or Run (easy/active recovery) | 30–45 min |

---

## Strength Session A — Upper Push + Legs

**Warm-up (5 min)**
- Ankle circles x 20 each direction
- Leg swings x 10 each
- Arm circles + shoulder rolls
- 5 min easy bike or jump rope

**Main (35–45 min)**
- Push-ups — 4 x max reps (add vest when easy)
- Ring Dips — 3 x 8–10
- Dumbbell Shoulder Press (20s or 25s) — 3 x 10–12
- Dumbbell Lateral Raises — 3 x 12
- Goblet Squat (25 lb) — 3 x 12
- Bulgarian Split Squat (bodyweight or vest) — 3 x 8 each leg
- Punching Bag — 3 x 2 min rounds

**Cool-down**
- Chest + shoulder stretch
- Hip flexor stretch
- Ankle mobility: heel drops, alphabet, resistance band circles

---

## Strength Session B — Upper Pull + Core

**Warm-up (5 min)**
- Ankle mobility (same as above)
- Cat-cow x 10
- Band pull-aparts x 15
- Light row warm-up

**Main (35–45 min)**
- Ring Rows — 4 x 10–12 (feet elevated when easy)
- Pull-ups / Band-assisted Pull-ups — 3 x max
- Dumbbell Bent-over Row (25 lb) — 3 x 10 each arm
- Ring Face Pulls (band or ring) — 3 x 12
- Plank — 3 x 45–60 sec
- Dead Bug — 3 x 10 each side
- Hollow Body Hold — 3 x 20–30 sec
- Dumbbell Bicep Curl — 2 x 12

**Cool-down**
- Lat stretch / child's pose
- Thoracic rotation
- Ankle mobility

---

## Run Sessions

### Easy Run (base rebuild)
- Current target: 3–5 miles at conversational pace
- No pace pressure — HR zone 2 (can hold a conversation)
- Road or trail, your call
- Post-run: log summary here via Hermes

### Long Run (weekend)
- Build to 6–8 miles over 6–8 weeks
- Trail preferred when weather allows
- Walk breaks are fine — priority is time on feet

---

## Bike Sessions (Echelon)
- Active recovery — 20–30 min easy spin
- Or substitute on rainy/cold days instead of outdoor run
- Log via Garmin/Echelon summary

---

## Progression Guidelines
- Weeks 1–2: Learn the sessions, focus on form, lighter weights
- Weeks 3–4: Add reps or small weight bump where comfortable
- Week 5+: Add weighted vest to bodyweight moves, extend runs
- Re-evaluate goals at the 6-week mark
