# 6-Day Bike Run Row Strength Plan

**Goal:** fat loss / body recomposition  
**Schedule:** flexible 6 days/week  
**Session length:** 45 minutes on weekdays  
**Rowing frequency:** 2 days/week  
**Run baseline:** 30-40 minutes continuous easy  
**Injury limits:** none currently noted  
**Equipment:** Concept2 rower, Echelon bike, running shoes, dumbbells, weighted vest, rings, bands, punching bag, Garmin

## Operating rules

- Six training days, one true rest day.
- Move sessions around as needed, but preserve the pattern: hard/easy alternation, 2 strength days, 2 rowing days, 2 running days, 1 bike day.
- Do not stack the hardest row, run, and strength sessions on consecutive days if avoidable.
- Most cardio should feel controlled. Fat loss comes from consistency plus enough easy volume, not from turning every session into a cage match.
- Warm up ankles/calves before runs and lower-body strength.
- If recovery feels poor, replace the bike or interval row with easy Zone 2 work.

## Weekly template

### Session 1 — Strength A + short easy cardio

**Intent:** full-body strength, muscle retention, fat-loss support.

- Warm-up: 5 min easy bike or row + ankle mobility
- Goblet squat or weighted vest squat: 3 x 8-12
- Push-ups or ring dips: 3 x controlled reps, stop 1-2 reps before failure
- Dumbbell shoulder press: 3 x 8-12
- Ring rows: 3 x 8-12
- Dead bug or plank: 3 sets
- Finish: 8-10 min easy bike/row/walk if time allows

### Session 2 — Easy run

**Intent:** aerobic base, calorie burn, running comeback.

- 30-40 min continuous easy run
- Conversational effort, mostly Zone 2
- Optional: 4 x 20 sec relaxed strides at the end if legs feel good
- Do not chase pace yet

### Session 3 — Pete Plan row: endurance / technique

**Intent:** rowing base and technical efficiency.

Start with the Pete Plan Week 1 endurance row, adapted if needed:

- 5000m easy/moderate row
- Focus: technique, relaxation, efficiency
- Stroke rate target: controlled, not frantic
- Log: distance, time, avg split, stroke rate, HR, feel

If 5000m is too much in the first week, cut to 2500-3000m and build from there.

### Session 4 — Strength B + loaded/carry bias

**Intent:** durable strength, upper body, posture, trunk.

- Warm-up: 5 min easy bike or row + band pull-aparts + ankle mobility
- Ring rows or band-assisted pull-ups: 4 x 6-12
- Dumbbell Romanian deadlift: 3 x 10-12
- Bulgarian split squat or reverse lunge: 3 x 8 each side
- Dumbbell row: 3 x 10 each side
- Weighted vest walk, farmer hold, or suitcase carry: 8-10 min total
- Optional finisher: punching bag 3 x 1 min easy/moderate rounds

### Session 5 — Indoor bike Zone 2

**Intent:** low-impact aerobic volume, recovery-friendly calorie burn.

- 40-45 min Echelon ride
- Effort: steady Zone 2, can talk in short sentences
- Cadence: comfortable and smooth
- Keep this boring on purpose. Boring works.

### Session 6 — Run or Pete Plan row intervals, alternating weekly

**Intent:** one controlled quality session per week.

Alternate each week:

#### Week A: run quality

- 10 min easy warm-up
- 6 x 1 min comfortably hard / 2 min easy jog or walk
- 10 min easy cool-down
- Keep reps controlled. Finish feeling like one more rep was possible.

#### Week B: Pete Plan row intervals

Use the corresponding Pete Plan beginner interval, starting with:

- 6 x 500m / 2 min rest
- First time: conservative pace, no hero rep until the final interval
- Log avg split and stroke rate

## Rest day

One full rest day per week. Walking is fine. No guilt cardio.

## Progression

### Weeks 1-2: establish rhythm

- Keep all cardio easy except Session 6.
- Strength: leave reps in reserve.
- Row: prioritize technique over split.

### Weeks 3-4: add small volume

- Easy run can move toward 40-45 minutes if recovery is good.
- Bike can stay 45 minutes or add slight resistance while staying Zone 2.
- Strength adds reps before load.

### Weeks 5-6: tighten quality

- Keep two easy cardio days truly easy.
- Improve row splits only if technique stays clean.
- Add weighted vest to squats/push-ups only if joints feel good.

## Weekly success standard

A winning week is not perfection. It is:

- 2 strength sessions completed
- 2 rowing touches completed
- 2 easy aerobic sessions completed
- At least 1 full rest day
- No injury flare-ups
