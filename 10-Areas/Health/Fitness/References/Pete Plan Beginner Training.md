# Pete Plan Beginner Training

**Source:** https://thepeteplan.wordpress.com/beginner-training/  
**Captured:** 2026-05-30

## Why this matters

The Pete Plan Beginner Training plan is a 24-week beginner-to-intermediate indoor rowing progression. It starts with 3 core rowing sessions per week and 2 optional rowing sessions, building volume gradually while using previous sessions to set realistic pacing targets.

For Michael, this is useful as a rowing backbone, but it should be adapted rather than copied straight because the target plan also needs indoor bike, running, and 2 strength days per week.

## Core principles to preserve

- Start easier than ego wants, especially in the first 2-4 weeks.
- Use prior workouts to set targets for future workouts.
- Build rowing volume gradually.
- Focus on technique, relaxation, and efficiency.
- Negative split when appropriate: start controlled, finish slightly stronger.
- Treat optional sessions as optional, not mandatory debt.
- Keep the training diary useful: log pace, stroke rate, feel, and anything affecting performance.

## Session types from the plan

### Endurance rowing

Examples: steady distance rows, 20-30 minute rows, 2 x 15 min, 3 x 10 min.

Purpose: aerobic base, technical efficiency, durability.

### Speed endurance / threshold rowing

Examples: 4 x 2000m, 5 x 1500m.

Purpose: improve sustained faster effort without turning everything into a race.

### Speed rowing

Examples: 6-8 x 500m, 4 x 1000m, 5 x 800m.

Purpose: 2k-style speed, higher breathing load, short controlled intervals.

## Adaptation idea for Michael

Do not run the full Pete Plan plus running, biking, and strength. That is too much too soon.

Better adaptation:

- Keep 2-3 rowing sessions per week.
- Make one row Pete Plan-style endurance.
- Make one row technique/easy aerobic.
- Make one row optional intervals or rotate it with bike/run intensity.
- Use the indoor bike for lower-impact aerobic volume or recovery.
- Keep runs mostly easy until the weekly habit is stable.
- Keep strength to 2 full-body days using dumbbells, rings, bands, vest, and bodyweight.
