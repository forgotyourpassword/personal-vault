## Quick capture
When something comes up, choose one:
- New actionable project → create a note from `Home/Templates/Project Template.md` in `Home/Projects/Active/`
- House knowledge / issue / measurements → add it to a note in `Home/Areas/`
- Vendor / contractor / quote source → create a note from `Home/Templates/Contractor Template.md`
- Purchase worth remembering → create a note from `Home/Templates/Purchase Template.md`
- Big decision → append to `Home/Decisions.md`

- Review active projects

### After work happens
Capture immediately:
- what changed
- who did it

- photos/receipts links

## Project sizing guide
Use these values consistently so future planning questions are easy to answer.

### Priority
How much it matters overall.
- low = nice to do
- medium = should get done
- high = important / keeps nagging
- critical = safety, damage, or major disruption

### Urgency
How soon it should happen.
- low = can wait
- medium = soon
- high = this week
- critical = now / before damage gets worse

### Level of effort
How hard the work is.
- tiny = 15-30 min
- small = 1-3 hours
- medium = half day
- large = full day
- huge = multi-day / contractor-scale

### Cost
Rough expected spend.
- low = under $100
- medium = $100-$500
- high = $500-$2,000
- major = $2,000+

## Principles
- Keep notes useful, not perfect
- One note per real project
- House knowledge lives in `Areas/`
- Decisions go in `Decisions.md`
- Use consistent priority / urgency / effort / cost labels
- Don’t build a giant system you won’t maintain
